Two ways to pin

macOS has no public way for one app to force another app's window to stay on top — so PinTop gives you the two honest alternatives, and lets you pick per pin.

Live mirror

Genuinely always on top

PinTop floats a live video copy of your chosen window above everything, at the window's real size. The original keeps keyboard focus — keep typing in it while the mirror stays visible. Perfect for a video call, a build log, or reference material. View-only, draggable, resizable.

Raise mode

The real, clickable window

PinTop asks macOS to re-raise the actual window whenever something covers it. You interact with the window itself — it just keeps coming back to the front as you switch between apps.

Questions

Why does PinTop ask for Accessibility or Screen Recording permission?

Raise mode uses macOS Accessibility to identify the focused window and ask the system to raise it — nothing else. The live mirror uses Apple's ScreenCaptureKit, which requires the Screen Recording permission to capture the one window you pin, only while it's pinned. PinTop is notarized by Apple and the source is public.

What happens when the trial ends?

Creating new pins pauses until you enter a license key. Nothing is deleted, nothing nags you on a timer — the menu simply shows a buy link and a key field.

Which Macs does it run on?

Apple Silicon Macs running macOS 13 Ventura or later. Raise mode works back to macOS 12.

Is the mirror interactive?

No — the mirror is a live, view-only copy. That's what lets the original window keep keyboard focus while the mirror stays on top. Click the visible edge of the original (or the mirror's source window) to interact with the real thing.
