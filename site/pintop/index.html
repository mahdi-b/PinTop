Two ways to pin

macOS has no public way for one app to force another app's window to stay on top — so PinTop gives you two honest alternatives, lets you pick per pin, and supports multiple simultaneous pins in any mix.

Live mirror

Genuinely always on top

PinTop floats a live video copy of your chosen window above everything, at the window's real size. The original keeps keyboard focus — keep typing in it while the mirror stays visible. Perfect for a video call, a build log, or reference material. View-only, draggable, resizable.

Raise mode

The real, clickable window

PinTop asks macOS to re-raise the actual window whenever something covers it. You interact with the window itself — it just keeps coming back to the front as you switch between apps.

Questions

Why does PinTop ask for Accessibility or Screen Recording permission?

Raise mode uses macOS Accessibility to identify focused windows and ask the system to raise them — nothing else. Live mirrors use Apple's ScreenCaptureKit, which requires Screen Recording permission to capture only the windows you explicitly mirror, only while they are pinned. PinTop is notarized by Apple and the source is public.

Will PinTop cost money later?

The planned price is a one-time $3.99 purchase with no subscription. The current release remains unrestricted; any transition in a later release will be explained before licensing returns.

Which Macs does it run on?

Apple Silicon Macs running macOS 13 Ventura or later. Raise mode works back to macOS 12.

Is the mirror interactive?

No — the mirror is a live, view-only copy. That's what lets the original window keep keyboard focus while the mirror stays on top. Click the visible edge of the original (or the mirror's source window) to interact with the real thing.

Why is it free right now?

The payment account is still awaiting approval. Rather than leave anyone locked out, trial enforcement and checkout are disabled until purchasing is ready.
